# Firebase Firestore Standard Edition Guide

The documentation describes a comprehensive skill for working with Cloud Firestore Standard Edition. Here are the key components:

**Overview**: This resource covers "provisioning, security rules, and SDK usage" for getting developers started with Firestore.

**Main Topics Covered**:

1. **Provisioning** - Referenced in provisioning.md for setting up Firestore in Firebase projects and local environments

2. **Security Rules** - Guidance on "writing and deploying Firestore Security Rules to protect your data" via security_rules.md

3. **SDK Usage** - Platform-specific implementation guides, with Web (Modular SDK) documentation available

4. **Indexes** - Information on index types, query support tables, and optimization practices

**Requirements**: While compatible with Firebase CLI (installed via `npm install -g firebase-tools`), the skill doesn't mandate its use.

The documentation is structured as a skill resource designed to help developers implement Firestore across their applications with proper security configurations and best practices.
